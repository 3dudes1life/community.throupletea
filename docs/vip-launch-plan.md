# VIP Launch Plan

## Phase 1 — Founding 50
Invite-only private beta. Build culture first.

## Phase 2 — Early Access 250
Founding members nominate trusted people. Expand slowly.

## Phase 3 — Public Waitlist
Applications open publicly while access remains curated.

## Phase 4 — Public Launch
Only launch broadly after culture, moderation, and platform structure are stable.
