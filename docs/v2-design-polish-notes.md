# V2 Design Polish Notes

This version fixes the typography and header readability across every page.

## Changes
- Header increased from cramped to breathable.
- Brand tagline changed to readable sentence case: "Built by us. For us. Always free."
- Navigation font weight reduced so it does not fight the logo.
- Navigation spacing improved.
- Hero headlines reduced and constrained for readability.
- Page hero typography balanced across all internal pages.
- Responsive header improved for smaller screens.
- Text wrapping improved with `text-wrap: balance` and `text-wrap: pretty`.

## Goal
Keep the premium dark neon Throuple Tea feel while making every page readable and polished.
