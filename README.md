# Throuple Tea Community Website

A GitHub Pages-ready starter site for **The Throuple Tea Community**.

## What is included

- `index.html` — main landing page
- `guidelines.html` — full community guidelines
- `join.html` — waitlist page
- `assets/style.css` — full styling
- `assets/script.js` — current year footer script
- `docs/community-blueprint.md` — deeper launch plan and moderation structure

## How to use with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from branch**.
5. Choose `main` and `/root`.
6. Save.

## Waitlist form

The join page currently uses a placeholder Formspree URL:

`https://formspree.io/f/YOUR-FORM-ID`

Replace that with your Wix Form, MailerLite, Brevo, Google Form, or other waitlist form action/link.
